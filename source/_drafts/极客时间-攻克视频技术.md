##### YUV

##### YUV的存储
+ Planar：
  + 先连续存储所有像素点的Y，然后接着存储所有像素点的U，之后再存储所有像素点的V。
  + 先连续存储所有像素点的Y，然后接着存储所有像素点的V，之后再存储所有像素点的U。
+ Packed：
  + 先存储完所有像素的Y，然后U、V连续的交错存储


| YUV类型     | 存储类型   |             |
|-----------|--------|-------------|
| YUV 4:4:4 | YV24   |
|           | I444   |
| YUV 4:2:2 | Packed | NV16        |
|           |        | NV61        |
|           | Planar | YU16 (I422) |
|           |        | YV16        |
|           |        |             |
| YUV 4:2:0 | Packed | NV12        |
|           |        | NV21        |
|           | Planar | YU12 (I420) |
|           |        | YV12        |


##### RGB和YUV之间的转换
Color Range：
+ Full Range：R、G、B取值范围都是0～255
+ Limited Range：R、G、B取值范围是16～235

转换标准：
+ BT709
+ BT601
